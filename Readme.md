# How to View Local Markdown Files in VS Code with Markdown Preview Enhanced

This guide explains how to open and preview local Markdown files in Visual Studio Code using the **Markdown Preview Enhanced** extension.

## 1. Install the extension

1. Open VS Code.
2. Go to the **Extensions** view by pressing `Ctrl+Shift+X`.
3. Search for `Markdown Preview Enhanced`.
4. Install the extension by **Yiyi Wang**.

## 2. Open your Markdown file

1. Open your project folder in VS Code.
2. Open any Markdown file, for example:
   - `week1.md`
   - `week2.md`

## 3. Open the enhanced preview

1. Press `Ctrl+Shift+P` to open the Command Palette.
2. Run the following command:

   `Markdown Preview Enhanced: Open Preview to the Side`

## 4. Make local images and files work correctly

Use relative paths in your Markdown files so that local images and linked files can be resolved properly.

## 5. Keep the preview updated

1. Save the file after making changes, or enable **Auto Save**.
2. The preview will refresh to reflect your latest updates.
